# Ski & Rescue 2026 App

A mountain safety and rescue management platform built for the 2026 ski season.

## Overview

Ski & Rescue 2026 is a Next.js web application designed for ski patrol teams to manage incidents, coordinate rescues, track personnel, and monitor trail status in real time.

## Features (planned)

- **Incident Dispatch** — Log, triage, and assign rescue teams to mountain incidents
- **Live Trail Map** — Real-time overlay of open/closed runs, patrol positions, and active incidents
- **Team Management** — Track patrol staff, certifications, and shift assignments
- **Audit & Reports** — End-of-day reports, compliance exports, and season analytics

## Tech Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
